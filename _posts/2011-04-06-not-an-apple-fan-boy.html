It was 2002 and with the purpose of learning new things I started using Linux. I was in love with open source software movement and I liked the fact that everything wall, well open, and extendible. As a computer science student it helped me by making me learn concepts that maybe using other environments (namely MS Windows). Basic networking, how a compiler works (the whole tool chain), basic operating system theory and even programming (via Perl). As a user I enjoyed by tweaking and playing with the system, configuring winmodems, installing the lasting distribution, submitting bugs and even doing some translation. As a developer I felt really comfortable using the GNU tools (grep, awk, sed, etc) and I still consider one of my assets as a developer.
